# deteccao-anomalias-xgb
Sistema completo de detecção de fraudes em cartões de crédito com ML/DL. 10 modelos, 76.8% F1-Score, 85.7% recall, R$ 51k evitados. Python, Scikit-learn, XGBoost, TensorFlow, Streamlit, FastAPI. Interface web interativa, API REST, SHAP, relatórios automáticos. Bootcamp Bradesco.
